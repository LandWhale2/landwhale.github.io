# 복잡한 템플릿 디버깅하기

렌나르트 레게브로(Lennart Regebro)가 추천하는 방법은 다음과같다.

템플릿들이 복잡해져서 어디서 변수들이 문제를 일으키는지 알수가없을때,

TEMPLATES 세팅의 OPTIONS 에서 string_if_invalid 옵션을 설정 함으로써 좀 더 자세한 에러 메시지를 받아볼수있다.

```json
#  settings/local.py

TEMPLATES = [
  {
    'BACKEND' : 생략,
    'APP_DIRS' : 생략,
    'OPTIONS':
      'string_if_invalid': 'INVALID EXPRESSION: %s'
  }
]
```
