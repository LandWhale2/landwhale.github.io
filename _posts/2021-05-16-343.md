# nginx 사용하는 웹서버 빨라지게 하기


```
location ~* \.(?:ico|css|js|gif|jpe?g|png)$ {
          expires 30d;
          add_header Vary Accept-Encoding;
          access_log off;
    }
```


사용하던 location 위에 이거 추가해주면 빨라짐
